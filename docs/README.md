## ✍️ Convenzioni di scrittura

Per uniformare la documentazione, ogni file Markdown dovrebbe avere:

- 📌 Un **titolo con emoji** in prima riga (`# 🎯 Titolo`)
- 📅 Una **riga di metadata** (versione + data aggiornamento), tipicamente:
  > Versione: 2.1.0 · Aggiornato: 2026-MM-DD
- 📚 Una sezione **🔗 Riferimenti** con link agli altri documenti correlati
- 🇮🇹 Lingua italiana (anche se il codice è in inglese)
